# Code of Conduct

## Our standards

This is a community resource list. We expect all contributors and participants to:

- Be respectful and constructive in issues and pull requests
- Focus feedback on content quality, accuracy, and relevance — not on personal opinions about projects or their authors
- Assume good faith when reviewing contributions

## What we don't accept

- Spam or self-promotional submissions without genuine community value
- Harassment or personal attacks of any kind
- Misleading descriptions or fabricated project details

## Reporting

If you see behavior that violates this code of conduct, open an issue or contact the maintainers directly.

## Scope

This code of conduct applies to all interactions in this repository — issues, pull requests, and any associated discussions.
