# Printify & Etsy Store Setup Checklist

Use this step-by-step implementation guide to link your Printify fulfillment account to Etsy or Shopify and configure automated fulfillment.

---

## 🛠️ Step-by-Step Store Launch Blueprint

### Step 1: Connect Store via OAuth Bridge
1. Log in to **Printify** -> Navigate to **My Stores** -> Click **Add New Store**.
2. Select **Etsy** (or **Shopify**). Authenticate your shop account via OAuth.
3. Grant Printify permission to sync product listings, inventory levels, and orders automatically.

### Step 2: Configure Automatic Order Routing
1. In Printify, navigate to **Store Settings** -> **Order Routing**.
2. Toggle **Enable Order Routing** to ON.
3. Select **Route Order to Secondary Supplier** if the primary print provider runs out of black size L t-shirts. This prevents canceled orders during holiday rushes.

### Step 3: Set Default Product Publishing Rules
1. **Mockup Selection:** Choose 2-3 lifestyle mockups (e.g., model wearing shirt in daylight) + 1 flat lay design graphic.
2. **Title & Tags:** Auto-sync primary search terms (e.g., *"Cyberpunk Cat Shirt, Japanese Synthwave Tee, Futuristic Graphic Shirt"*).
3. **Variants:** Select standard sizes (S, M, L, XL, 2XL) in high-demand colors (Black, Dark Grey, Navy, White, Athletic Heather).
