# Fashion Lookbook Moodboard & Lighting Specs

Use this template to standardize the visual style, studio lighting, and color settings for fashion editorial lookbooks.

## 1. Visual Vibe & Aesthetic Archetype
* **Theme Name:** e.g., Nordic Minimalist Autumn
* **Core Palette:** Warm taupe, soft sand, muted olive, slate grey.
* **Mood Keywords:** Editorial, quiet luxury, organic, architectural.

## 2. Studio Lighting Coordinates
* **Primary Key Light:** Large diffused softbox from the left at a 45-degree angle.
* **Fill Light:** White reflector bounce from the right to soften neck shadows.
* **Backlight/Rim:** Subtle warm backlight to separate model's shoulders from the background.
* **Ambient Lighting:** Soft overcast daylight.

## 3. Environment Backdrop Presets
* [ ] Pure white seamless cyclorama
* [x] Textured plaster screen in warm grey
* [ ] Architectural concrete columns
* [ ] Mid-century modern wooden partition screen

## 4. Prompt Tokens Matrix
Combine these tokens to generate lookbook backgrounds:

```
[Model Archetype] + [Garment Descriptors] + [Studio Lighting Specs] + [Camera Settings]
```

* **Model Archetype:** `Editorial fashion model standing in a relaxed pose,`
* **Garment Descriptors:** `wearing a minimalist neutral-toned organic knit sweater,`
* **Studio Lighting Specs:** `diffused side studio softbox lighting, soft neutral shadows,`
* **Camera Settings:** `shot on medium format Hasselblad 80mm lens, f/4, crisp details, natural skin textures.`
